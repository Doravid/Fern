Cute hackathon project I made with my best friend [Thanh](https://github.com/avoforme)! The concept is a puzzle game where the puzzles are language! Since Thanh is vietnamese we decided to teach vietnamese with the game. We made the puzzles very quickly with no play testers so some of them are kind of really difficult... but overall I actually really like the idea! The inspiration was [Chants of Sennaar](https://store.steampowered.com/app/1931770/Chants_of_Sennaar/). All the art was done by me! I'm not really an artist but I had fun making the scenes. The Banh Mi cart is my favorite. I also did most of the backend graphics stuff since I had used openGL before. The puzzles were made by Thanh, as well as the Dictionary you see on the right. Thanh made the whole dictionary system super flexible, so its possible to add in language support for both input, and output language. Part of our pitch was that this game could be expanded a LOT. I think it could be. I really like the idea to be honest.  

You might see this game runs on the browser, see below, and think we are using wasm, or some other cool technology to port the game to the web! The answer is: **NO**. This game is actually just a website. Its built entirely in JS and HTML. The graphics are rendered using WebGL and the text is rendered using HTML elements. Input handlings is just js input listeners. 

Did I mention we chose this tech stack at a Hackathon? This wasn't just any Hackathon though. It was HackMIT. One might think that MIT would have a good hackathon, but like, it was 90% AI grifters. Both the sponsors and the other students. I think one of the sponsors was bragging about pulling a crypto rug pull during the opening ceremony. Why is any of this relevant? It's not really, but it did mean that Thanh and I wanted to go as far away from AI as would could. So we built a game with nothing but WebGL and a box of scraps. We were very locked in and built the entire thing without even really going on line to google things. The graphics libraries are taken from my computer graphics class which is why there is a bunch of functions for 3D.
