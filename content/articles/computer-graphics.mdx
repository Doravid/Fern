## Assignment 1: SVG Editor

The first assignment was about making a simple SVG loader, viewer, and editor. This was a pretty good introduction because it introduced us to the idea of coordinate systems. We needed to convert between world space and canvas space. Essentially, the SVGs has one set of coordinates, think -0.5 to 0.5 for x and y, with the center at (0,0), and the canvas had another, 0 to 500 for x/y with the top left being (0,0). 

![picture of the editor](/photoshop.png)
In order to simply transformations, like panning, zooming, and rotating, we used transformation matrices. This are super powerful because you can apply them to one another and the resulting matrix will apply both operation. E.g you can create a matrix to zoom and pan and rotate all at the same time.
