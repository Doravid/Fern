People say that this class should be skipped. They are wrong. This class is **fuego**.
![fire](/fire.jpg)
No really, this class was fun! It had a very standard structure for a math class at WPI. One or two online homeworks each week. A mid term, a final, a few written assignments scattered about. 

The professor for this class was an absolute G. He liked [The Case of the Golden Idol](https://store.steampowered.com/app/1677770) games, so he must be smart. 

I will admit that some of the content was a bit... light. Boolean logic is pretty easy but it was a good refresher. If nothing else, this class is Maths Math class. It fills in a lot of the gaps in math education. Those big and scary math symbols that your Calculus 3 loves to use? Probably taught in this class. Thats the general vibe of this class. 

It went over some pretty cool stuff. The basics of set theory which was fun! It also went over relations and functions, which I don't really remember how exactly they work, but I remember having fun learning about them! 

We also learned counting!! This is not a joke.

## Conclusion

This class had it all: 
* It was Fun
* It had easy grading
* My discussion leader was super nice
* The professor loved the subject and you could tell

Overall, immaculate vibes. I'm glad I took this class even if it did replace a 4000 computer science class. If I didn't take it, I would have taken a painful 4000 class like Databases 2 or something annoying like that.
